https://moneynoteee.netlify.app/

## 사용 기술
* 리액트
* 타입스크립트
* Sass

<br/>

## 구현 기능
* 내역 추가: 해당 버튼을 누르고 항목 입력 후 등록
* 달력 기능
    * 내역 추가 중 날짜 입력 시
    * 상단 월 선택 버튼 클릭 시
* 수입 / 지출 분류
* 내역 수정: 아이템 클릭 후 수정버튼 누르고 항목 수정
* 내역 삭제: 아이템 클릭 후 삭제버튼 클릭
